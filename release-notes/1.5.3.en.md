## v1.5.3

### AI 助手接管更多报错 / AI Assistant Handles More Failures

- ComfyUI 启动失败之外，单个插件安装失败、单个 Python 包安装失败，现在也可以直接拉起 AI 助手继续排查和修复。  
  In addition to ComfyUI startup failures, the AI Assistant can now also take over single plugin install failures and single Python package install failures.

- 同一轮修复里的多步操作，授权体验更清楚，减少反复打断。  
  Multi-step repair flows now feel smoother, with clearer approval behavior and fewer repeated interruptions.

### 插件安装更顺 / Better Plugin Installation

- 安装插件时，会先查本地插件信息；找不到时再联网搜索仓库地址。  
  Plugin install now checks local plugin data first, then falls back to online repository lookup when needed.

- 像 `InstantID` 这类插件，比以前更容易找到正确仓库并继续安装。  
  Plugins like `InstantID` are now easier to resolve to the correct repository and continue installing.

### 积分与登录体验改进 / Better Credits and Sign-In Experience

- AI 会话顶部现在可以直接看到积分，扣分或退回积分后会自动刷新。  
  You can now see your credits directly in the AI chat header, and the balance refreshes automatically after charges or refunds.

- 设置页和 AI 会话里的积分显示已经对齐，不再一个变了另一个没更新。  
  The Settings page and AI chat now stay in sync, so credit balance no longer updates in only one place.

- 未登录时，用户仍然可以先输入消息；发送时会先拉起登录，登录成功后继续发送。  
  When signed out, users can still type first; sending will prompt sign-in, then continue with the original message after login.

### 依赖下载修复 / Dependency Download Fixes

- 修复了 7-Zip 下载异常问题，替换了已失效的镜像下载源。  
  Fixed 7-Zip download failures by replacing a broken mirror source.

- 如果下载仍然失败，现在会更明确显示已经尝试过哪些下载地址。  
  If the download still fails, the app now shows which download sources were attempted.

### 感谢赠送 / Thank-You Credits

- 已向现有注册用户发放一次性 `30` 积分。  
  A one-time `30` credit gift has been issued to existing registered users.

- 这批赠送积分有效期为 `180` 天。  
  These bonus credits are valid for `180` days.
