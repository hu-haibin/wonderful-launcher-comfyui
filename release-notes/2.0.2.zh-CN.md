## v2.0.2

### WinUI 自动更新

- WinUI 现在会在启动后自动运行静默更新链路，不再只依赖设置页里的手动检查。
- 桌面应用可以在后台准备更新内容，并在更新就绪后给出重启提示。
- 更新状态判断更稳，不容易再出现误导性的降级提示或陈旧状态提示。

### 中文界面恢复

- 恢复了 WinUI 主启动器的 `跟随系统 / 简体中文 / English` 语言选择能力。
- Home、Settings、启动状态、更新提示和最常用的启动器主路径现在会重新跟随所选语言。
- 发版仓 README 也补上了当前 WinUI 实机路径的最新界面截图。

### 发版链路加固

- 新增了基于本地假更新源和隔离 profile 的 Windows 自动更新 smoke 流程。
- 发版收尾现在会在发布前验证真正的根入口 `ModelFinder.exe`，以及 `Home -> 启动 -> 内嵌 WebView` 这条真实用户路径。
