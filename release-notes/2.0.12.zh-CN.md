# ModelFinder 2.0.12

发布于 2026 年 5 月 18 日。

这个版本重点改善 WinUI Agent 修复链路、修复过程可见性、环境刷新防卡死，以及 Agent 反馈数据的隐私边界。

## 更新内容

- **启动失败时 Agent 接管更直接**：当 ComfyUI 启动失败时，WinUI 会把失败证据交给 Agent 修复流程，而不是让用户自己复制日志或反复手动重启。
- **修复过程更可见**：Agent 面板会说明当前正在检查什么、准备修改什么、执行后会如何验证，不再只是展示一串工具名称。
- **降低 UI 等待拖死修复的风险**：环境刷新和工具后上下文刷新增加了安全超时，避免某个 UI 刷新太慢时把 Agent 工具链一直卡住。
- **依赖修复更实用**：WinUI 环境页仍保护 `pip`，但不再阻止 PyTorch 相关包进入有意替换/修复的流程。
- **Agent 反馈上下文已做隐私约束**：用户主动提交 Agent 反馈时，桌面端可以上传经过脱敏和长度限制的对话摘要，用于改进后续诊断质量。
- **Avalonia 经典界面恢复**：Avalonia 桌面壳已恢复到接近 1.5.3 的旧版界面，方便仍依赖这条构建路径的用户。

## 建议更新人群

如果你使用 WinUI 桌面版，并且依赖 Agent 启动修复、依赖修复、缺失节点排查或反馈上报，建议更新到这个版本。
