## v2.0.3

### WinUI Chinese Feature Pages

- Restored localized text across the main WinUI management surfaces instead of leaving large feature pages half-English.
- Deploy, Environment, Plugin Manager, Model Manager, Model Finder, and Download Center now respect the launcher language setting.
- This release is focused on the feature pages users stay in after startup, not only the Home and Settings shell.

### Better Public Presentation

- The release repository README now includes a feature gallery with fresh screenshots from the current WinUI public build.
- The gallery now shows the launch surface, environment view, plugin manager, model manager, workflow model finder, and download center.
