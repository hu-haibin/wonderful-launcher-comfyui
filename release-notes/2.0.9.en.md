# ModelFinder 2.0.9

Released on May 13, 2026.

This release restores the public Windows installer to a self-contained package. It is the corrective release for the 2.0.7/2.0.8 packaging regression that could show .NET runtime prompts on machines without the matching desktop runtime.

## What's Fixed

- **Self-contained installer restored**: the Setup Installer now carries the app runtime payload again, so normal installs should not require a separate Microsoft .NET Desktop Runtime install.
- **No runtime prerequisite dialog in the installer path**: the installer no longer bundles or launches a .NET Desktop Runtime prerequisite flow for the normal public release.
- **Release artifacts remain installer-only**: the public release assets are the Setup Installer and `SHA256SUMS.txt`; no portable zip is published for the normal release.
- **Release guardrails tightened**: tests and the release skill now block accidental packaging-shape changes such as switching between self-contained and framework-dependent without explicit approval.
