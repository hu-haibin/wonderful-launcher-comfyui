# ModelFinder 2.0.11

Released on May 16, 2026.

This release is a focused stability update for the WinUI desktop app. It improves the task terminal and cleans up several user-facing messages, especially around startup, Agent, Image, Photoshop, update, and close flows.

## What's New

- **Early ComfyUI output is preserved**: if runtime logs arrive before the task terminal is fully open, ModelFinder now keeps those bytes and replays them into the terminal.
- **Cleaner terminal progress**: launcher status lines and raw runtime output no longer interfere with each other's terminal control sequences as easily.
- **Clearer stop and cancel results**: stopping ComfyUI from the task terminal is less likely to leave a confusing completed/canceled state.
- **Better localized desktop copy**: more WinUI dialogs, Agent prompts, Image actions, Photoshop messages, and update/close prompts now use the proper language resources.
- **More reliable packaged lookups**: model-source lookups use the packaged JSON contract more consistently in the desktop build.

## Who Should Update

Update if you use the WinUI desktop build, especially if you rely on startup logs, task terminal output, Agent-assisted repairs, or bilingual UI prompts.
