# ModelFinder 2.0.14

发布于 2026 年 5 月 19 日。

这个 hotfix 主要处理真实用户反馈中的任务终端未响应、插件安装、部署完成流程和大参考图上传问题。

## 更新内容

- **UI 挂死更容易远端诊断**：新增后台 watchdog，在诊断允许时，即使 UI 线程完全卡住，也能记录并上报便于聚合统计的 `ui_thread_unresponsive` 事件。
- **运行时停止状态更可靠**：插件修复或 Agent 依赖安装停止 ComfyUI 时，启动器会退出旧的 Running/workspace 状态，回到可恢复的空闲状态。
- **任务终端证据更清楚**：插件安装终端会记录会话和子进程结果，子进程非零退出码会显示为警告，不再误报成成功。
- **部署完成更顺手**：欢迎页内联部署完成后会自动注册新包并启动 ComfyUI，去掉多余的一次“启动 ComfyUI”点击。
- **部署进度文本不再重叠**：进度百分比和当前解压步骤改为独立列显示，长步骤名会自动省略。
- **大参考图上传更稳**：超大的参考图上传前会自动压缩，减少 413 请求体过大的失败。

## 建议更新人群

如果你会通过启动器安装插件或依赖、在欢迎页部署 ComfyUI、使用 Agent 修复环境，或者在图片工作流里上传大参考图，建议更新到这个版本。
