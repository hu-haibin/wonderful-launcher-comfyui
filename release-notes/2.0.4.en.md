## v2.0.4

### Image Workspace to Photoshop is now a smoother main flow

- `Send to Photoshop` is now the default one-click handoff from generated images.
- If Photoshop is connected but no document is open, ModelFinder now creates a new document automatically before importing the image.
- If a document is already open, the image is sent into the current document as a new layer instead of forcing a separate export step.
- Sending or importing no longer tears down the result surface just to show an error; the image stays visible while the app shows sending, success, or recovery feedback.

### Clearer send controls in the workspace and viewer

- The main result area now uses a clearer `Send to Photoshop` label instead of exposing Photoshop document details too early.
- A secondary `Send as New PS Doc` action is still available for people who want an explicit new-document branch.
- History sends now share the same busy-state guard as the main result view, which reduces accidental double sends.

### Verified real-machine flow

The release evidence for this version includes a real `generate -> send to Photoshop` pass with both Photoshop branches:

- Photoshop connected with zero open documents
- Photoshop connected with an existing document
