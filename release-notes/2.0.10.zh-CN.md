# ModelFinder 2.0.10

发布于 2026 年 5 月 14 日。

这个版本重点补上桌面端转化遥测和任务终端稳定性，同时继续沿用 2.0.9 已恢复的 self-contained Windows 安装包形态。

## 更新内容

- **Agent 和图片生成转化埋点**：桌面端现在会记录 Agent 使用与图片生成的关键漏斗事件，包括请求、确认使用、完成、失败，以及登录用户的积分余额检查点。
- **WinUI 与 Avalonia 分开标记**：遥测事件会带上桌面 UI 家族，后续可以分别观察 WinUI 与 Avalonia 的使用情况，而不是混在一起看。
- **扩展商业事件上报**：Agent 与图片生成转化事件会走已登录用户的商业遥测链路，方便后端统计更精确的转化数据。
- **任务终端输出更稳定**：WinUI 任务终端改为向 xterm bridge 转发原始 PTY 字节，改善进度输出和非 ASCII 终端内容显示。
- **下载和安装进度收尾更稳**：插件安装与 Nunchaku wheel 下载在成功、失败或取消时都会正确结束终端进度行。
- **遥测隐私加固**：诊断遥测值里如果包含 Windows 本地路径片段，会在离开客户端前脱敏。
