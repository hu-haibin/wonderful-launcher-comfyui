# ModelFinder 2.0.14

Released on May 19, 2026.

This hotfix focuses on real-world stability reports around task terminals, plugin installation, deployment completion, and large reference-image uploads.

## What's New

- **UI freezes are now easier to diagnose remotely**: a background watchdog records full UI unresponsive periods and sends aggregate-friendly `ui_thread_unresponsive` telemetry when diagnostics are enabled.
- **Runtime stop state is cleaned up more reliably**: when plugin repair or Agent dependency tools stop ComfyUI, the launcher now leaves stale Running/workspace states and returns to an idle, recoverable state.
- **Task terminal evidence is clearer**: plugin install terminals now record session/process outcomes, and non-zero subprocess exits are shown as warnings instead of false success.
- **Deployment completion is smoother**: inline deployment automatically registers the deployed package and starts ComfyUI after completion, removing the extra manual launch click.
- **Deployment progress text no longer overlaps**: the progress percentage and current extraction step now use separate layout columns with truncation for long step names.
- **Large reference images upload more safely**: oversized reference images are compressed before upload to reduce 413 request-size failures.

## Who Should Update

Update if you install plugins or dependencies from the launcher, deploy ComfyUI from the welcome page, use the Agent for environment repair, or upload large reference images for image workflows.
