# ModelFinder 2.0.10

Released on May 14, 2026.

This release focuses on decision-grade desktop telemetry and task terminal stability while preserving the self-contained Windows installer shape restored in 2.0.9.

## What's New

- **Agent and Image conversion telemetry**: the desktop app now reports key funnel events for Agent usage and Image generation, including request, accepted, completed, failed, and credit-balance checkpoints for signed-in users.
- **WinUI and Avalonia are tagged separately**: telemetry events now include the desktop UI family so WinUI adoption and Avalonia usage can be compared without mixing the two clients.
- **Commercial-event forwarding expanded**: Agent and Image conversion events are sent through the same authenticated commercial telemetry path used by the website, giving the backend cleaner conversion metrics.
- **Task terminal output is more reliable**: WinUI task terminals now forward raw PTY bytes to the xterm bridge, improving progress output and non-ASCII terminal rendering.
- **Download and install progress cleanup**: plugin install and Nunchaku wheel download progress now finalize terminal frames correctly on success, failure, or cancellation.
- **Telemetry privacy hardening**: diagnostic telemetry values now redact embedded Windows paths before they leave the client.
