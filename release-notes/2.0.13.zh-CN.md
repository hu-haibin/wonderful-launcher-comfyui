# ModelFinder 2.0.13

发布于 2026 年 5 月 19 日。

这个版本清理了工具页未接通的占位入口，继续打磨部署/导入路径，并修正了一些 WinUI 外壳体验细节。

## 更新内容

- **工具页不再展示未完成动作**：移除了“批量分析”“批量下载”“AI 自动修复”三个占位卡片，等对应流程真正接通后再开放。
- **已有实例的导入入口更容易找到**：WinUI 部署界面在检测到已有 ComfyUI 实例时，仍会保留“导入”入口。
- **版本选择更清楚**：版本选择、刷新、打开 release 和回退文案路径更稳定，并补齐了本地化。
- **窗口和底栏体验打磨**：标题栏显示应用版本，主题色切换可在当前界面刷新，底栏按钮 hover 尺寸更一致。
- **降低 GitHub release API 压力**：release 元数据会被缓存，并通过镜像回退减少重复请求。

## 建议更新人群

如果你使用 WinUI 版部署或导入 ComfyUI 环境，或者之前觉得工具页占位入口容易误导，建议更新到这个版本。
