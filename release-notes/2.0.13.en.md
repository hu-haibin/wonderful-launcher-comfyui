# ModelFinder 2.0.13

Released on May 19, 2026.

This release cleans up unfinished Tools-page entries, improves the deployment and import path, and smooths several WinUI shell details before the next public build.

## What's New

- **Tools page no longer advertises unfinished actions**: Batch Analysis, Bulk Download, and AI Auto-Fix placeholder cards were removed until those workflows are fully wired.
- **Existing-instance import is easier to find**: the WinUI deploy surface keeps the Import action visible when existing ComfyUI instances are detected.
- **Release selection is clearer**: version selector, refresh, open-release, and fallback text paths are steadier and localized.
- **Window and bottom-bar polish**: the title bar now shows the app version, theme brush changes refresh in place, and bottom-bar hover sizing is more consistent.
- **Lower GitHub release API pressure**: release metadata is cached and mirror fallback avoids repeated request bursts.

## Who Should Update

Update if you use the WinUI build to deploy or import ComfyUI environments, or if the old Tools-page placeholders made the feature surface look more complete than it really was.
