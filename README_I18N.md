# Wonderful Launcher quick download notes

[English](README.md) | [简体中文](README_CN.md) | 繁體中文 | 日本語 | 한국어

This page is a short multilingual download note. The main maintained README files are English and Simplified Chinese.

## Universal download note

- Latest release: <https://github.com/hu-haibin/wonderful-launcher-comfyui/releases/latest>
- Download the Windows installer named `WonderfulLauncher-Setup-v...exe`.
- Do not download GitHub's auto-generated `Source code.zip` or `Source code.tar.gz`.
- The installer is for Windows 10/11.
- Checksums are published as `SHA256SUMS.txt` in the same release.
- Official website: <https://wonderfullauncher.com/>
- Docs: <https://wonderfullauncher.com/docs>

## 繁體中文

Wonderful Launcher 是 Windows 上的 ComfyUI 桌面啟動器。它可以匯入或部署 ComfyUI、查看啟動日誌、處理缺失模型與節點，並提供受控的修復流程。

請前往最新版 Release，下載 `WonderfulLauncher-Setup-v...exe`。不要下載 `Source code.zip` 或 `Source code.tar.gz`，那些不是可直接執行的安裝包。
